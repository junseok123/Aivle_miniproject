# Aivle_School_miniprojcet
## 수어 GPT 번역 서비스
수어로 ChatGPT를 사용하는 웨 서비스
개발기간 2023 - 05 - 17 ~ 2023 - 05 - 25

## 프로젝트 소개
1. ChatGPT
   - ChatGPT API 연동하여 채팅의 기능을 구현
2. Signlanguage to Chatgpt
   - ChatGPT API 연동하여 수어 사진 업로드 시 번역 모델 구현
